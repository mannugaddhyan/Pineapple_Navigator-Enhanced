Mitigation

ID Mitigation Description
M1048 Application Isolation and Sandboxing Consider utilizing seccomp, seccomp-bpf, or a similar solution that restricts certain system calls such as mount. In Kubernetes environments, consider defining a Pod Security Policy that limits container access to host process namespaces, the host network, and the host file system
M1038 Execution Prevention Use read-only containers, read-only file systems, and minimal images when possible to prevent the running of commands
M1026 Privileged Account Management Ensure containers are not running as root by default and do not use unnecessary privileges or mounted components. In Kubernetes environments, consider defining a Pod Security Policy that prevents pods from running privileged containers
