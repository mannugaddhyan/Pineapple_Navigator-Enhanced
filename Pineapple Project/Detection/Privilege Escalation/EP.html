ID Data Source Data Component Detects
DS0027 Driver Driver Load Look for behavior on the endpoint system that might indicate successful compromise, such as abnormal behavior of the processes. This could include suspicious files written to disk, evidence of Process Injection for attempts to hide execution or evidence of Discovery. Consider monitoring for the presence or loading (ex: Sysmon Event ID 6) of known vulnerable drivers that adversaries may drop and exploit to execute code in kernel mode
DS0009 Process Process Creation Monitor for newly executed processes that may exploit software vulnerabilities in an attempt to elevate privileges.
